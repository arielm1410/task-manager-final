# Task Manager Pro - אפליקציית ניהול משימות 

אפליקציית מנהל משימות מתקדמת ומעוצבת שנבנתה כפרויקט מסכם בקורס React. האפליקציה מאפשרת למשתמשים לארגן את המשימות שלהם ביעילות, עם תמיכה בקטגוריות צבעוניות, תאריכי יעד וממשק משתמש מודרני.

##  תכונות מרכזיות
* **ניהול משימות מלא:** הוספה, עריכה, מחיקה וסימון משימות כהושלמו.
* **קטגוריות דינמיות:** יצירת קטגוריות חדשות עם בחירת צבעים מתוך פלטה מעוצבת.
* **תאריכי יעד:** הגדרת דד-ליין למשימות באמצעות לוח שנה.
* **תצוגה גמישה:** מעבר בלחיצת כפתור בין "תצוגת רשימה" (List) ל"תצוגת כרטיסיות" (Grid).
* **סינון וחיפוש:** סינון משימות לפי סטטוס (הכל/לביצוע/הושלמו) וחיפוש חופשי בטקסט.
* **שמירה אוטומטית:** כל הנתונים נשמרים ב-LocalStorage של הדפדפן.

##  טכנולוגיות
* **React 19**
* **Vite**
* **Tailwind CSS v4**
* **Lucide React** (Icons)
* **Vitest & React Testing Library** (Bounus Tests)

##  הוראות הרצה (Installation)

1.  התקן את התלויות:
    ```bash
    npm install
    ```

2.  הרץ את שרת הפיתוח:
    ```bash
    npm run dev
    ```

3.  **הרצת בדיקות (בונוס):**
    הפרויקט כולל בדיקות יחידה (Unit Tests) לרכיבים המרכזיים. להרצה:
    ```bash
    npm test
    ```

##  רשימת קומפוננטות ואחריותן

1.  **`App`**: הקומפוננטה הראשית - מנהלת את ה-State, השמירה ל-LocalStorage והלוגיקה המרכזית.
2.  **`TaskForm`**: טופס להוספת משימות ויצירת קטגוריות חדשות.
3.  **`TaskList`**: אחראית על רינדור רשימת המשימות וניהול התצוגה (Grid/List).
4.  **`TaskItem`**: מציגה משימה בודדת ומטפלת בעריכה, מחיקה ושינוי סטטוס.
5.  **`FilterBar`**: רכיב סינון המשימות לפי סטטוס (הכל/פעיל/הושלם).
6.  **`SearchBar`**: שורת חיפוש לסינון לפי טקסט.
7.  **`Footer`**: מציג מונה משימות וכפתור לניקוי משימות שהושלמו.

##  מגבלות ידועות
* המידע נשמר ב-LocalStorage (דפדפן) ולא מסתנכרן לענן.
